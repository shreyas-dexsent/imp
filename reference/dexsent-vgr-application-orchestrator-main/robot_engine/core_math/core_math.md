# core_math

Mathematical base for transforms, rotations, Lie groups, interpolation,
numerical derivatives, optimization utilities, metrics, and tolerances.

Conventions:

- 4x4 homogeneous matrices for transforms.
- Right-handed frames.
- Quaternion ordering is `[x, y, z, w]`.
- Twist ordering is `[vx, vy, vz, wx, wy, wz]`.
